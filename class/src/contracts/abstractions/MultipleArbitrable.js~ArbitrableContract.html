    ArbitrableContract

    Extends:

    AbstractContract → ArbitrableContract

    Arbitrable Abstract Contarct API. This wraps an arbitrable contract. It provides interaction with both the off chain store as well as the arbitrable instance. All arbitrable methods from the supplied contract implementation can be called from this object.
