    Kleros

    The Kleros Api provides access to the full suite of functionality. It will initialize contract instances for you when possible and creates an object that you can use to call all of the other api modules. If you are only going to be interacting with specific apis, or you don't want certain functionality such as the off chain store, you might find it easier to initialze a specific instance of the api you want.

    public constructor(ethereumProvider: string, storeUri: string, arbitratorAddress: string, arbitrableContractAddress: string) source

    Instantiates a new Kelros instance that provides the public interface to Kleros contracts and library. All params are required. To use an individual portion of the API import a class and initialize it yourself.

    Params:

    NameTypeAttributeDescription
    ethereumProvider string

    The Web3.js Provider instance you would like the Kleros.js library to use for interacting with the Ethereum network.
    storeUri string

    The storage provider uri used to get metadata from the cloud for the UI. e.g. Kleros-Store, IPFS, Swarm etc.
    arbitratorAddress string

    Address of the arbitrator contract we should use when initializing KlerosPOC
    arbitrableContractAddress string

    Address of the arbitrator contract we should use when initializing KlerosPOC
